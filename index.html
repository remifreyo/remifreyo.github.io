Amazon Clone

This project is a clone of the Amazon web store. It allows users to sign up for an account, add products to their shopping cart, and checkout using Stripe payment processing.

Tech Stack Utilized: ReactJS, NodeJS, Stripe API, Firebase, Material UI and Vanilla CSS.

This project is deployed on Firebase.

QuizApp

QuizApp is a web app that allows users to take quizzes, store their results and retake previously taken quizzes to generate new results.

Tech Stack Utilized: ExpressJs, EJS, NodeJS, OAuth, Vanilla CSS, MongoDB, and Mongoose.

QuizApp has CRD functionality and is deployed on Heroku.

Happenings

Happenings is a web app that was built with 2 other members of my cohort at General Assembly. It has full CRUD capabilities allowing users to create, update, and delete events, and add and remove vendors to created events.

Happenings leverages Python and the Django framework at its core and uses the Django Template Language for front end rendering. PostgreSQL was chosen as our database of choice.

Happenings is currently deployed on fly.io.
